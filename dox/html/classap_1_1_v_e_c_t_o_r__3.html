Detailed Description

template<typename real>
class ap::VECTOR_3< real >

Vector class.

VECTOR_3 originates as part of the Config Lib for Panukl software: https://itlims-zsis.meil.pw.edu.pl/software/PANUKL/2020/Config_API/index.html created by. F.A.Dul, modif. by T.Grabowski.
The VECTOR_3 class is header only part of the ap library. It means that only the header file: ap_vector.h has to be included in the project and linking is not needed.
Implementation of the VECTOR_3 class aims at universality and ease of use. It is small, easy to understand and with clean code sytax resembling as close as possible equations notation.

◆ SetGolobalResizeFlag()

template<typename real >
void ap::VECTOR_3< real >::SetGolobalResizeFlag ( const bool  val)
inline

Set global resize flag.

Size of the VECTOR_3 is compared with the input data. If they don't match the Set() function may return failure value. The behavior depends on the flags set:

  • grFlag - global resize flag
  • rFlag - local resize flag of the particular Set() operation

Set() methods perform boolean operations as in the following table:

grFlag grFlag Result
1 1 1
1 0 0
0 1 0
0 0 0

The table indicates that resizing of the VECTOR_3 with Set() methods will be possible only when both flags are true.

Note
By default both resizing flags are true.
Parameters
val- value witch is set to the grFlag

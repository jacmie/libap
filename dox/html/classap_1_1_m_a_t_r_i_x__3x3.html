◆ SetGolobalResizeFlag()

template<typename real >
void ap::MATRIX_3x3< real >::SetGolobalResizeFlag ( const bool  val)
inline

Set global resize flag.

Size of the MATRIX_3x3 is compared with the input data. If they don't match the Set() function may return failure value. The behavior depends on the flags set:

  • grFlag - global resize flag
  • rFlag - local resize flag of the particular Set() operation

Set() methods perform boolean operations as in the following table:

grFlag grFlag Result
1 1 1
1 0 0
0 1 0
0 0 0

The table indicates that resizing of the MATRIX_3x3 with Set() methods will be possible only when both flags are true.

Note
By default both resizing flags are true.
Parameters
val- value witch is set to the grFlag

Detailed Description

Parser for substituting (inserting, or deriving) values of variables in files.

ap::SUBSTITUTE object has to be initiated with input file, output file, and unique marks indicating variables.
Ending mark can be a whitespace, but it is advised to use explicit mark.
Next map of variables is added and finally variables are inserted to the output file, or extracted from the output file. See examples in tests.

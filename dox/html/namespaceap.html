Arc aproximation with bezier curve.

Third order (4 points) bezier curve aproximates arc. The arc has to have less than 90deg.

Parameters
x1- first point x coordinate, beggining of the arc and the bezier curve
y1- first point y coordinate, beggining of the arc and the bezier curve
x2- returned second point x coordinate
y2- returned second point y coordinate
x3- returned third point x coordinate
y3- returned third point y coordinate
x4- returned fourth point x coordinate, end of the arc and the bezier curve
y4- returned fourth point y coordinate, end of the arc and the bezier curve
xCirc- circle center x coordinate
yCirc- circle center y coordinate
Returns
0 on success

Detailed Description

NACA airfoils class.

Generates 4-digit and 5-digit naca airfoils.
ap::NACA_AIRFOIL originates from part of the Config Lib for Panukl software: https://itlims-zsis.meil.pw.edu.pl/software/PANUKL/2020/Config_API/index.html created by T.Grabowski.

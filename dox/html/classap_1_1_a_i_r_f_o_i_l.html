Detailed Description

Airfoils class.

ap::AIRFOIL class can read geometry data from file (5 commonly used fromats), or generate 4-digit and 5-digit NACA airfoils.
ap::AIRFOIL originates from part of the Config Lib for Panukl software: https://itlims-zsis.meil.pw.edu.pl/software/PANUKL/2020/Config_API/index.html created by T.Grabowski.

void ap::AIRFOIL::RestrictiveCheck ( bool  setFlag = 0)
inline

Sets flag for restrictive check of data when read.

Additionaly data end values are checked. Values different then 0 and 1 for Xfoil format and 0, 100 for Prf format trigger failure while read.

Parameters
setFlag- flag for restrictive data checking, default false
